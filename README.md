# tbkmini-col2row-zmk

This Firmware assumes that you use this PCB https://github.com/Bastardkb/Dactyl-Manuform-PCB-Plate

```
PCB Pin | MCU Pin
R2      | 4
R3      | 5
R4      | 6
R6      | 7
```

```
PCB Pin | MCU Pin
C1      | 19 (only used for TBKmini, not for Sekeltyl)
C2      | 18
C3      | 15
C4      | 14
C5      | 16
C6      | 10
```

```
Thumb PCB Pins
C2, C4, C5, R6
```
